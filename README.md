# README #

WEB is a static single page app repository that is responsible to access the Marvel API.

### WEB responsabilites and operations ###

* centralize all Marvel interfaces in a single project
* handle all routes

### Normal Flow ###

* npm start

### Libs and Frameworkss ###
* Angular 1.6
* JSDocs
* Eslint
* Jest
* Gulp
